## 📖 bilibili-FOCUS

看 Bilibili 总是因为大数据推送分心？
相关推荐一看就停不下来？  

这个插件帮你屏蔽掉冗余的推荐内容，让你专注于视频和搜索本身，并支持在指定时段或条件下强制保持开启。`v3.0` 进一步把锁定规则、播放结束推荐屏蔽和扩展内界面重绘整合成一套更完整的专注体验。

---

### 🔥 v3.0 重点升级
- **锁定策略正式成型**
  - 新增每周固定时段锁定、专注倒计时锁定、每日累计使用时长锁定
  - 命中任一规则时，四项核心功能都会被强制开启，无法从 popup 关闭

- **播放页去干扰能力合并**
  - 自动关闭「自动连播」
  - 视频结束后自动隐藏播放器内相关推荐，减少被下一条内容继续牵走

- **控制面板与设置页全面重绘**
  - popup 与设置页统一为更紧凑的专注式布局
  - 支持浅色 / 深色主题
  - 新增独立设置页，用于维护锁定规则与界面主题

---

### ✨ 功能介绍
- **全局专注重定向 (New!)**
  - 自动拦截容易让人分心的首页、动态流等页面，直接重定向到纯净搜索页
  - 内置智能白名单，安全放行视频播放、个人空间、消息中心、账号设置等核心功能
  
- **搜索页优化**  
  - 使用 [https://search.bilibili.com](https://search.bilibili.com) 获得纯净的搜索框页面  
  - 隐藏顶部导航栏和个性推送  
  - 搜索框居中显示，界面简洁无干扰  

- **播放页优化**  
  - 播放页只保留播放器区域  
  - 通过遮罩降低对比度和聚焦光晕效果，视觉更舒适，让你专注于视频内容 
  - 自动关闭播放页的「自动连播」，并隐藏播放结束后的相关推荐，避免视频结束后继续被下一条带走

- **锁定策略**
  - 支持每周固定时段锁定，在设定时间内强制保持插件功能开启
  - 支持手动开启专注倒计时，在倒计时结束前禁止关闭功能
  - 支持按前台可见使用时长累计，超过每日阈值后锁定到下一个重置时间
  - 锁定生效时，四项功能都会被自动强制开启，无法从弹窗关闭

- **控制面板**  
  - 提供锁定状态、四项功能主开关和专注倒计时快捷操作
  - 详细规则通过独立设置页维护，避免弹窗过度拥挤
  - 弹窗与设置页统一重绘，支持浅色/深色主题和更紧凑的专注式布局

### 🛠️ 安装方法

1. 下载或克隆本仓库：
   ```bash
   git clone https://github.com/Silhouette-my/bilibili-FOCUS.git
   ```
2. 打开 Chrome 浏览器，进入扩展管理页面：
   ```
   chrome://extensions
   ```
3. 打开右上角 **开发者模式**  
4. 点击 **加载已解压的扩展程序**  
5. 选择本仓库目录，即可启用插件

### 🚀 使用说明

- 安装完成后，浏览器工具栏会出现插件图标  
- 点击图标打开控制面板，可以自由切换功能开关  
- 需要长期约束时，可在设置页配置固定时段、每日累计时长和界面主题
- 在搜索页和播放页体验纯净界面，避免推荐干扰
